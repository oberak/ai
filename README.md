### Structures of Directory

- /src: Python sources(*.py), Jupyter notebook files(*.ipynb)
- /docs: Document files(*.md)
- /members/{id or name}/**: 개인 파일
  + /members 폴더 아래 개인폴더를 하나 만드시고 그아래에 개인 파일을 올리시면 됩니다
